# spice-parser
A simple parser for SPICE format.

Packages: 
	- flex v2.6.0 
	- bison v3.0.4 
	- make v4.1 
	- gcc v5.4.0 

Compile: 
	Just execute 'make' in the main directory. A binary file will be generated in the 'bin' directory. 

Usage: 
	'./bin/spice-parser [file]'
